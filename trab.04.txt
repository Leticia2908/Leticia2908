Python é uma linguagem de programação interpretada, conhecida por sua simplicidade e legibilidade. É amplamente utilizada em várias áreas, como desenvolvimento web, ciência de dados, automação de tarefas, entre outros.

JavaScript:
JavaScript é uma linguagem de programação interpretada, principalmente usada para criar páginas web interativas. É executada no lado do cliente (navegador) e também no lado do servidor (Node.js).
console.log("Olá, mundo!");
