Inteligência artificial - é um conceito de programação na área da computação que representa a capacidade de criar algoritmos para que as máquinas possam interpretar dados e aprender com essa interpretação, usar o aprendizado para resolver determinadas tarefas e se comportar de maneira muito semelhante à forma como os humanos agem, como pensar, como analisar, raciocinar, como aprender e tomar decisões de maneira lógica.                                                         
Dados - os dados são a base da informação. Ele não conseguia descrever completamente a situação. Pode ser quantificado, mas não qualificado. Precisa- se dos dados para obter informação.             
Banco de dados- Um banco de dados é uma coleção organizada de informações ou dados estruturados, geralmente armazenados eletronicamente em um sistema de computador. Os bancos de dados geralmente são controlados por um sistema de gerenciamento de banco de dados (SGBD).                            
Servidor - Um servidor é um computador ou sistema de computação que responde a solicitações de dispositivos clientes em uma ou mais redes (locais ou remotas) e é capaz de executar aplicações (programas), fornecendo capacidades de processamento e/ou armazenamento de dados.                    
Memória - Memória é um dispositivo que tem a função de armazenar dados na forma de sinais digitais por um determinado período de tempo. Existem dois tipos de memória: RAM e ROM. | RAM (Random Access Memory) é a memória que permite gravar e reescrever dados, mas se o computador for desligado, por exemplo, as informações gravadas são perdidas. A memória ROM (memória somente leitura) permite que os dados sejam gravados uma vez, o que significa que nenhuma informação pode ser excluída ou editada, apenas ela pode ser acessada.
